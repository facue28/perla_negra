Guía Visual de Carga de Productos

Utilice esta guía para entender qué campo del Excel modifica cada parte de la página web.

Nota: Para guardar esto como PDF, presione Ctrl + P y seleccione "Guardar como PDF".

1 image_url La imagen principal del producto.

3 name Nombre principal del producto.
4 price Precio numérico (usar punto para decimales).

7 size / code / sensation Datos técnicos que aparecen en la caja de información.
